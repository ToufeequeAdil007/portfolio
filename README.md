## ✨ A Premium, AI-Powered Personal Portfolio Website

A futuristic portfolio website for **Toufeeque Adil** - AI Engineer, AI Agent Developer, and Modern Web Developer. Features an intelligent AI chatbot powered by Groq's Llama 3.3 model.

![Portfolio Preview](https://via.placeholder.com/800x400?text=Toufeeque+Adil+Portfolio)

## 🚀 Features

### 🤖 AI-Powered Chatbot
- Real-time AI responses using Groq Llama 3.3
- Voice input support
- Chat history and memory
- Constrained to portfolio-only questions
- Beautiful glassmorphism UI

### 🎨 Premium Design
- Glassmorphism effects
- Particle.js animated background
- Fully responsive (Mobile, Tablet, Desktop)
- Smooth scrolling animations
- Professional typography

### 📧 Contact System
- Premium email templates
- Auto-reply to visitors
- Professional design for both admin and user

### 📁 Project Showcase
- 7+ featured projects
- Modal popup with project details
- GitHub links
- Category filtering

## 🛠️ Tech Stack

### Frontend
- HTML5, CSS3, JavaScript
- Particles.js for animations
- Font Awesome icons
- Google Fonts (Inter)

### Backend
- Node.js & Express.js
- Groq SDK (Llama 3.3 70B)
- Nodemailer for emails
- CORS enabled
